This repository contains the complete solutions for the **DBMS Laboratory (18IS507)**. It includes Schema designs (DDL) and complex Query implementations (DML) for 5 core database scenarios as per the lab manual.

## 🚀 Project Overview

- **Database Engine:** MySQL / Oracle
- **Key Concepts:** ER-to-Relational Mapping, Joins, Nested Queries, Aggregate Functions, and Views.
